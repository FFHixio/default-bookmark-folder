Quick bookmark icon options

This feature is optional.

It will add a quick bookmark icon in the address bar, to (un)bookmark a page with only one click. The icon is a round star icon in the address bar. If the current page is bookmarked, the icon color will change to red. Here are two screenshots to serve as examples: when the current page is not bookmarked and when it is.
